# InvestigacionDPS_02
Creacion de un sistema de realidad aumentada para un panel que contiene la temperatura y la humedad 

## 🛠️ Guía de Instalación
1. Clonar el repo.
2. Ejecutar `npm install`.
3. Para iniciar: `npx expo start` (o el comando que usen).

## 🧪 Pruebas de QA Realizadas

- Renderizado 3D: ✅ Optimizado en formato GLTF.
